# 전등배치 원클릭 앱

KS A 3011 조도 등급표 + 전등 수량 계산 + 다실 일괄 계산 + AutoLISP(`QQQ`, `QQQALL`) 코드 생성 기능을 제공하는 웹앱입니다.

## 기능

- **KS A 3011 조도 등급(A~K)**
  - 최저 / 표준 / 최고 선택 시 조도(Lx) 자동 반영
- **단일 실 계산**
  - 면적, 층고, 광속, 보수율 기반 추천 수량 계산
- **다실 일괄 계산**
  - CSV 형식 입력으로 실별 RI/U/추천 수량 자동 계산
- **AutoLISP 생성**
  - 현재 설정값을 기본값으로 가진 `qqq_auto.lsp` 생성/다운로드
  - `QQQ`: 단일 실(경계 생성)
  - `QQQALL`: 다중 닫힌 경계 일괄 배치

## 계산식

- 유효 높이: `Hm = (층고 - 850) / 1000`
- 실지수: `RI = 면적 / (Hm × (가로m + 세로m))`
- 이용률(U):
  - `RI >= 4.0 → 0.80`
  - `RI >= 2.5 → 0.75`
  - `RI >= 1.5 → 0.65`
  - `RI >= 1.0 → 0.55`
  - 그 외 `0.45`
- 추천 수량:
  - `N = ceil((목표조도 × 면적) / (광속 × U × 보수율))`
